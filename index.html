Web-framework for Ruby.

Web framework, wich uses a lot of Ruby magic to achive nice balance between code readability, productivity of developer and speed of execution (but speed of execution is least important)

How to use it?

Extend your controller with Dhh::Controller. And you will get the route called https://base_url/controller_name/action_name

class HelloController < Dhh::Controller # this will be https://base_url/hello/index def index render :index, noun: :unicorn end

# dump_controller which show rack content def env_info ar = env.to_s.split(',') output = '' ar.each do |line| output += "#{line.to_s}" + "<br>" end output end end
